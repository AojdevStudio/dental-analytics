## Solution Summary

### Problem Fixed
The dashboard was crashing with an `ImportError` because `chart_production.py` was trying to import four helper functions that didn't exist in `chart_utils.py`:
- `add_trend_line_to_figure`
- `add_pattern_annotation` 
- `format_currency_hover`
- `handle_empty_data`

### Solution Implemented
Added the missing helper functions to `apps/frontend/chart_utils.py`:

1. **`add_trend_line_to_figure`** - Wrapper for the existing `add_trend_line` function to maintain backward compatibility
2. **`add_pattern_annotation`** - Wrapper for the existing `add_trend_pattern_annotation` function  
3. **`format_currency_hover`** - Enhanced currency formatter with negative value and `None` handling
4. **`handle_empty_data`** - Creates a clean empty state chart with informative message

### Additional Fixes
- Fixed test imports in `test_chart_integration.py` and `test_plotly_charts.py`
